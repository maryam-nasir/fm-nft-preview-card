### What I learned

While working on this project, I learnt how to group hover on elements using Tailwind CSS. A simple code snippet is as follows:

```
<div class="relative group">
    <div class="absolute hidden group-hover:block">
        <!-- Child 1 -->
    </div>
    <div class="absolute hidden group-hover:block">
        <!-- Child 2 -->
    </div>
    <div class="absolute hidden group-hover:block">
        <!-- Child 3 -->
    </div>
</div>
```
